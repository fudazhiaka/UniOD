# UniOD: A Universal Model for Outlier Detection across Diverse Domains

Official implementation of UniOD: AbUniversal Model for Outlier Detection across Diverse Domains from The Fourteenth International Conference on Learning Representations (ICLR 2026). Includes the code for training and evaluating UniOD.

# Setup Instructions
You will need to install the packages in the requirement file for this project. To install all packages, run the following command:
```
pip install -r requirements.txt
```
